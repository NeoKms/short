# Project rules

- JSON API использует `snake_case`.
- Ошибки API имеют форму `{"error":{"code":"...","message":"..."}}`.
- Короткий код состоит из 8 символов base62 и генерируется криптографически безопасно.
- Разрешены только абсолютные URL с `http` или `https`, без credentials, длиной до 4096 символов.
- PostgreSQL является источником истины; сбой Redis при чтении не должен ломать редиректы.
- Новые изменения схемы добавляются отдельным SQL-файлом миграции; существующие миграции после релиза не редактируются.
- Секреты передаются только переменными окружения или GitHub Secrets.
